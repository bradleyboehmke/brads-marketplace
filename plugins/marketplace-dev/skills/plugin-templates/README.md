# Plugin Templates Skill

This skill provides comprehensive templates for creating plugin components.

## Purpose

Provides ready-to-use templates for:
- Agent files
- Command files
- Plugin documentation

## Contents

- **agent-template.md** - Complete template and guidelines for creating agents
- **command-template.md** - Complete template and guidelines for creating commands

## When to Use

Load this skill when:
- Creating new plugin components
- Need structural guidance
- Want to ensure consistency
- Teaching plugin development

## Quick Reference

### Agent Template

```markdown
---
description: Brief description (5-10 words)
---

You are a [Name], an expert in [domain].

## Your Expertise
- [Capability list]

## Your Approach
1. [Step-by-step methodology]

## Key Principles
- [Core values and guidelines]
```

### Command Template

```markdown
---
description: Brief description (5-10 words)
---

[Context paragraph]

## Your Task
[Clear purpose statement]

## Step-by-Step Process
### Step 1: [Name]
[Instructions]
```

## Usage

Refer to the detailed template files for complete guidance, examples, and best practices.
